# ECShopX

## 项目简介

ECShopX 是一个基于 PHP Lumen 框架的电商系统后端，提供完整的 API 接口。

## 快速开始

**推荐**：查看完整开发与部署指南 → [docs/DEVELOPMENT_AND_DEPLOYMENT.md](docs/DEVELOPMENT_AND_DEPLOYMENT.md)

包含：
- 本地开发环境搭建
- 后端 + Admin 前端 + Desktop 前端联调
- Docker 容器化部署
